# Future FileLocker Features

## Privacy And Encryption

- Encrypted vault bundles for grouping related files into one portable `.locked` package.
- Keyfile generator with checksum display and printable backup guidance.
- Password strength scoring that stays local and never sends text off-device.
- Optional tamper-evident history log when Incognito Mode is off.
- Local-only secure share package that includes encrypted files, hashes, and a manifest.

## Secure File Handling

- Verified decrypt report showing restored filenames, hashes, and output locations.
- Sidecar checksum files for any encrypted or decrypted batch.
- Safer large-folder preview with file counts, total size, and extension mix before processing.
- Quarantine review area for files staged before secure deletion.
- Secure free-space wipe report with drive, method, pass count, and completion status.

## Usability

- Reusable local profiles for common encryption, hashing, and cleanup workflows.
- Import/export for FileLocker settings as a local JSON file.
- Keyboard-friendly command palette for major actions.
- Better batch failure recovery with retry only failed items.
- Compact operation summary that can be copied for personal records.

## Maintenance

- Cleanup preview diff before deleting temporary files.
- Browser cleanup profiles with clear separation between cache, cookies, and history.
- Drive optimizer status history that records only local drive health metadata.
- Registry cleanup backup manager for reviewing and restoring exported `.reg` backups.
